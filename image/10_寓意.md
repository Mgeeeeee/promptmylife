# 寓意
直接根据以下要求生成插画，不要提问，不要解释

## 具体任务
- 深度阅读理解文稿
- 提炼文稿内容核心
- 选择合适视觉意图
- 枯燥信息寓意于物
- 绘制生动形象插画

## 阅读文稿
- 注意时间维度或因果链条
- 提炼清晰、是非分明的中心主张
- 提取可对比的路径或视角
- 生成短标签和简短的结语

## 视觉意图
- 以「能最清楚表达文章因果与对比」为最高优先级
- 请自行发掘视角，若无合适视角，再从意图库挑选

## 意图库
包含但不止以下视角
- 循环 / 跃迁
- 强度 / 一致
- 短期 / 长期
- 即时 / 延迟
- 试错 / 成功
- 输入 / 输出
- 可见结果 / 隐性积累
- ……

## 寓意于物
- 生硬教条 → 生动形象
- 平铺直叙 → 自然规律
- 符号结构 → 万物生灵

## 插画风格
- 画面：白底、黑线、扁平、矢量、留白

- 强调色
+ 正向/积极/新生 → 天青色(#8dd4e8)
+ 负面/消极/陈旧 → 山茶红(#DC5B6F)

- 辅助色
+ 框架、文字 → 黑白
+ 点缀、过渡 → 莫兰迪

- 元素：简笔、图标、自然、生活
- 排版：对称、线性、循环、分区、对比
- 标注：标题置顶、结语置底、标注标签
- 语言：English

## 输出
- 直接生成图片（含标题与标签），不要多余解释
